# MyKoreanDish

## Description
This is a korean dish & ingredient review web application that lets users to review the taste, price, dietary and ingredient for dish and nutrient for ingredient so that people who have no idea about it gets an idea and it allows users to make it personalized by having personal notes and favorite dishes and ingredients.

## Features

- Google authentication
- Dish and ingredient search, filter and review
- Favorite dish and ingredient management
- Similar dish recommendation
- User activity history
- Review insights
- Users profile information
- Personal Note on each dish and ingredient

## Tech Stack

- **Backend**: Flask
- **Frontend**: Jinja2
- **Database**: Firebase
- **Authentication**: Google Auth
- **Deployment**: Render
